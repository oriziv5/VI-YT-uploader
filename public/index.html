How to use?

  1. Get your account id from the environment you want to upload to. you can find the account id at the account list or in your account settings
  2. Get your API Key or the account access token (allowEdit=true). If you aquire API Key make sure you take it from the right APIM environment. (prod)
  3. Choose if you want to upload batch videos (via a text file containing direct links seperated by line break) or one by one
  4. Apply other settings if you chose to upload just one video.
